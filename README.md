# E-Commerce Microservices Backend

## Overview

This project is a backend application built using Java and Spring Boot following the Microservices architecture. It is being developed as a learning project to gain practical experience with backend development and service-based applications.

## Technologies Used

* Java 17
* Spring Boot
* Spring Data JPA
* PostgreSQL
* Spring Cloud Eureka
* OpenFeign
* Maven
* REST APIs
* Git & GitHub

## Services

### Product Service

* Add Product
* View All Products
* PostgreSQL Integration
* REST API

### Order Service

* Currently under development
* Service-to-service communication using OpenFeign
* Eureka Client integration

## Features

* RESTful APIs
* PostgreSQL database connectivity
* Microservices architecture
* Maven project structure
* Service Discovery using Eureka

## System Architecture

Client → API Gateway → Eureka Server → Product Service → PostgreSQL

## Features

- Spring Cloud Gateway routing
- Service discovery using Eureka Server
- REST APIs for Product Service
- Load-balanced microservice communication
- PostgreSQL integration

  ## API Endpoints

GET /products → Fetch all products

## How to Run

1. Start Eureka Server
2. Start Product Service
3. Start API Gateway
4. Open http://localhost:8080/products
